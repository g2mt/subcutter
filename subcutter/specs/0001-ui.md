# Subtitle cutter

This is a subtitle cutting program written in PySide2. It will allow the user to cut a video by removing lines from a subtitle file.

It will have the following resizable panels (using QSplitter):

Left panel (default 80% width):
  - A widget containing all the subtitle lines, similar to a rich text box. The subtitle lines will be implemented by another widget. This widget can be highlighted similarly to a text line.

Right panel containing 2 resizable sub panels:
  - A widget for playing the current video.
  - A container widget with the following inputs:
    - Label "Input video:" in one line, then an input box
    - Label "Subtitle file (.srt):" in one line, then an input box
